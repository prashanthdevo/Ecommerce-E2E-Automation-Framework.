### 1. Test Plan Overview
* **Scope:** Login, Product Search, Cart Management, and Checkout.

* **Types of Testing:** Functional, Regression, and Smoke Testing.

### 3. Bug Reporting Template
I use a standard professional format for bugs:
* **Summary:** [Module] - Brief description of the issue.
* **Steps to Reproduce:** Detailed 1, 2, 3 steps.
* **Expected vs Actual:** Clear comparison.
* **Severity/Priority:** Critical/High/Medium/Low.
